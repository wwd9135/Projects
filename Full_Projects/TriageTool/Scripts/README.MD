# Overview
PowerShell scripts designed to gather windows forensic data
Designing output to be in JSON, so my python file can parse the data as a dict and feed it into an LLM for further analysis.

## Navigation
- Heuristic triage: A comprehensive forensic report of a windows device
- Network_Triage: A detailed network forensic analyis of a windows device
- Persistence_Triage: A detailed persistence forensic analysis tool, checking for an attackers attempts to establish persistence and control over a windows device.
- C2_Triage: A tool to scan for command and control server involvement on a device suspected to be compromised.
